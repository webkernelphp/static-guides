# Webkernel Documentation

**Follow the numbers. You will never get lost.**

This documentation uses explicit numeric prefixes (`00-`, `01-`, `02-` ...) at every level.

- Top level sections: `01-`, `02-`, ...
- Inside sections: `01-`, `02-` for individual pages

This guarantees order for humans, VitePress, other SSGs, and AI agents.

## Why numbered prefixes?

- No ambiguity about where to start or what comes next.
- Perfect for static site generators (VitePress, etc.).
- Great for AI agents and future tooling.
- Because this documentation lives inside a Composer package, it can be served directly from a developer panel (with playgrounds, resources, etc.) in the future.
